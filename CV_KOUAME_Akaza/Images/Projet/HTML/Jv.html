    Mon 2ème Hobbie...

    Les jeux-vidéos

    Cela pourrait vous paraître normal car les jeux-vidéos représentent grandement notre génération. Cependant, à l'inverse de ce que pensent beaucoup d'adultes, ce n'est pas juste un passe-temps inutile. Ce "passe-temps" peut rapporter de l'argent de différentes manières : en faisant des vidéos dessus, en jouant à haut niveau dans des compétitions (l'E-sport), en finissnat le jeu le plus rapidement possible (le speedrun)...

    De plus, il existe énormement de genres dans les jeux-vidéos



    Exemple :
    • Les FPS, abréviation de “First Person Shooter”(Counter Strike, Call of Duty, ... )
    • Dans la catégorie des FPS, nous avons les Batlle Royal(Fortnite, Apex Legends, ... )
    • Les RPG, abréviation de “role playing game”(Zelda, Final Fantasy, ... )
    • Les jeux de plateforme (Super Mario Bros, ... )
    • Les MOBA, pour “multiplayer online battle arena”(League of Legends, ... )
      • ...
